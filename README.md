# Exemplo de Node.js em App Engine flexible com profiler

Exemplo de aplicação Node.js com  App Engine flexible e profiler

## Setup

Antes de fazer o deploy

1.  Instalar as dependências:

    Com `npm`:

        npm install

    ou `yarn`:

        yarn install

## Running locally

Com `npm`:

    npm start

ou `yarn`:

    yarn start

## Deploying to App Engine

Com `npm`:

    npm run deploy

ou `yarn`:

    yarn run deploy
